Parliament Calendar Subscription

  • What is this?
  • How do I use it?
  • Who made it?

What is this?

Currently the only way to find the Parliament Calendar is calendar.parliament.uk. Many MP staff members spend time copying the entries over to their MP's calendars manually.
This tool provides a subscribable calendar which updates daily with the latest published deadlines.
Anyone is welcome to use it - the code is all public so you can even check how it works.



How do I use it?

With your calendar tool (probably Outlook), you can subscribe to these calendar urls:

Commons Calendar

https://storage.googleapis.com/hackathons-data/ToryTechs-Feb2020/commons.ics

Lords Calendar

https://storage.googleapis.com/hackathons-data/ToryTechs-Feb2020/lords.ics
To get setup, please follow the "Workaround" instructions here.



Who made it?
